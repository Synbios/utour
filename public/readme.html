两侧水平滑动目录UI

插件出处: http://plugins.adchsm.me/slidebars

源文件地址: vendor/assets/stylesheets/slidebars.css 和 vendor/assets/javascripts/slidebars.min.js

使用方法如下:

  1. html文件头加入 meta viewport 设定
  2. 主页内容放在#sb-site里面
  3. 目录内容放在.sb-slidebar里面
  4. 用.sb-left和.sb-right区分左右目录
  5. 添加打开/关闭目录的控制键（.sb-toggle-left 和 .sb-toggle-right）
  6. 添加jquery, sidebars.css和sidebars.min.js（以及front30.css.scss）
  7. javascript call $.slidebars()
      <!doctype html>
      <html>
      <head>
      <title>标题</title>
      <meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no">

      <!-- 引入 CSS -->
      <link rel="stylesheet" href="path/to/slidebars.min.css">
      </head>

      <body>
      <div id="sb-site">
      <!-- 主页内容 -->
      </div>

      <div class="sb-slidebar sb-left">
      <!-- 左目录 -->
      </div>

      <div class="sb-slidebar sb-right">
      <!-- 右目录 -->
      </div>

      <!-- 引入 jQuery -->
      <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

      <!-- 引入 js -->
      <script src="path/to/slidebars.min.js"></script>
      <script>
      (function($) {
      $(document).ready(function() {
      $.slidebars();
    });
  }) (jQuery);
  </script>
  </body>
  </html>

注意: 手机页面左右角按钮的css在app/assets/front30.css.scss里面

前端黑底糖果框架

本界面以bootstrap和jquery为基础整合了多个js插件,其中css的代码在app/assets/stylesheets/flatMania-css里面, js代码在app/assets/javascripts/flatMania-js. 建议把他们全部添加。由于这些jquery插件的功能比较复而这里就不单独介绍，具体说明请google检索各自的帮助文件。这里主要对几个使用了的界面做说明。

关于方块拼图的说明

方块拼图主要用于首页, 所对应的‘静态’文件是app/views/front30/home.html.erb. （erb文件是嵌入代码的html文件). 方块分为文字和图片和幻灯片。排版使用bootstrap默认的12个格子（cell）为一行(row)的手机、平板、网页一体式栅格模型。其中文字方块的格式如下：


  <!-- 外框 -->
  <div class="box box-md br-green animated">
  <!-- 鼠标进入方格后出现的半透明罩 -->
  <div class="box-hover box-hover-black">
  <!-- 半透明罩上德图标 -->
  <span class="hover-icon">进入</span>
  </div>
  <!-- 点击后的链接 -->
  <a href="/front30/group">
  <!-- 链接内容 -->
  <div class="box-content navigation">
  <!-- 方格上的图标 -->
  <i class="fa fa-users fa-5x"></i>
  <!-- 方格上德文字标签 -->
  <span class="box-title" style="font-size: 33px">跟团游</span>
  </div>
  </a>
  </div>

说明: 方块目前默认两种大小即中号(box-md)和大号(box-lg),大号的宽度是小号的2倍数。当然你也可以通过width和height手动改变方块的长宽。

说明: 方块有几种默认颜色分别是 br-white br-black br-green br-red br-yellow br-blue br-brown br-lblue br-pink。其他颜色可以用background-color设定。

图片方块于文字方块相似，只是内容略有不同

  <!-- 外框 -->
  <div class="box box-md animated">
    <div class="box-content box-gallery padd-zero">
      <!-- 图片所带链接 -->
      <a href="http://erp.utourworld.com/admin/ba/visa/visaSearchMobile.asp">
        <!-- 链接内容（和文字方格中得图标，标题类似） -->
        <div class="box-content navigation">

          <!-- 标题 -->
          <span class="box-title" style="font-size: 33px">签证</span>
        </div>
      </a>
      <!-- 背景图片 -->
      <img src="/images/front30_2.jpg" class="img-responsive box-gallery-img" alt="" />
    </div>
  </div>

前端动态目录js插件(isotope)

该插件用于跟团游对团队的过滤。其中地点标签的过滤用的是isotope (代码地址: public/javascripts/jquery.isotope.min) 日期的过滤用的javascript在页面内（app/views/front30/group.html.erb）, 日历插件的代码在public/javascripts/bootstrap-datepicker.js, public/javascripts/bootstrap-datepicker.zh-CN.js 和public/stylesheets/datepicker3.css

基本步骤

  1. 引入上述的js文件
  2. 入下图用javascript激活
<div id="container"> # 盒子
  <div class="item">...</div> # 物品标签
  <div class="item">...</div> # 物品标签
  <div class="item">...</div> @ 物品标签
  ...
</div>

js激活代码

$('#container').isotope({
  // options
  itemSelector : '.item', // 效果应用于该标签
  layoutMode : 'straightDown' // 布局样式
});

注意: isotope提供几种布局方式, 我们使用的是straightDown

通过javascript可以实现过滤功能

<div id="container">
  <div class="element a b">只显示a</div> # 添加标签 a b
  ...
</div>

..

// 过滤（只显示） 'a'
$('#container').isotope({ filter: '.a' }); // 过滤后只剩下标签a的物品

说明: 其他功能请参考官网
